# FocusFlow

FocusFlow is a clean and intuitive productivity web app designed to help you stay focused, track tasks, and boost efficiency with a distraction-free workflow.


## Features

-  Add, edit, and delete tasks  
-  Focus timer (Pomodoro style)
-  Meditation Sessions (With Various Themes)
-  Productivity analytics/report  
-  User accounts & login system
-  Maintain Journals

## Tech Stack

| Frontend | Backend | Database |
|--------|--------|--------|
| React / Vite | Node.js / Express | MongoDB |


##  Installation & Setup

### git clone https://github.com/yourusername/focusflow.git
### cd focusflow

## Setup Backend
### cd backend
### npm i
### Setup the .env file with MONGO_URI, JWT_SECRET, PORT and VITE_API_BASE
### node index.js

## Setup Frontend
### cd frontend 
### npm i
### Setup .env file with VITE_API_URL
### npm run dev
